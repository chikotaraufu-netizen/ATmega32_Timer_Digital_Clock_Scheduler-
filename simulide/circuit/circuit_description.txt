# SimulIDE Circuit — ATmega32 Timer-Based Digital Clock & Task Scheduler

## File
- **Circuit file:** `ATmega32_Clock.simu`
- **Firmware HEX:** `../../firmware/hex/main.hex`

## How to Run

1. Open SimulIDE
2. File → Open → Select `ATmega32_Clock.simu`
3. Right-click the ATmega32 chip → **Load Firmware** → Navigate to `firmware/hex/main.hex`
4. Set MCU frequency to **8 MHz** (right-click → Properties)
5. Click the **Play** button (▶) to start simulation

## Circuit Components

| Component | SimulIDE ID | Description |
|-----------|------------|-------------|
| ATmega32 MCU | mega32-1 | Main microcontroller, 8 MHz |
| LED (Yellow) | LED_STATUS | Status LED on PB0 — toggles every 2s |
| LED (Green) | LED_TASK | Task LED on PB1 — flashes every 5s |
| R3 (220Ω) | R3_220 | Current limiter for Status LED |
| R5 (220Ω) | R5_220 | Current limiter for Task LED |
| SET Button | SET_BTN | Push button on PA0 (key: S) |
| INC Button | INC_BTN | Push button on PA1 (key: I) |
| RESET Button | CLOCK_RST | Push button on PA2 (key: R) |
| R2, R4, R6 (10kΩ) | Pull-ups | External pull-up resistors for buttons |
| Serial Monitor | Serial_Monitor | USART output at 9600 baud on PD1 |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| **S** | Press SET button (enter/cycle time-set mode) |
| **I** | Press INC button (increment selected field) |
| **R** | Press RESET button (reset clock to 00:00:00) |

## Expected Behavior

1. **Startup:** Clock shows `00:00:00` on serial monitor
2. **Counting:** Seconds increment every 1 second
3. **Status LED (Yellow):** Toggles ON/OFF every 2 seconds
4. **Task LED (Green):** Flashes briefly every 5 seconds
5. **SET button:** Cycles through Hours → Minutes → Seconds → Normal
6. **INC button:** Increments the currently selected time field
7. **RESET button:** Resets clock to 00:00:00
