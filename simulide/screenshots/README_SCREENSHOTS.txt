# SimulIDE Screenshots
#
# Place your simulation screenshots here after running the simulation.
#
# Required screenshots (per project spec):
#
# 1. startup.png        - System at startup showing 00:00:00
# 2. counting.png       - Clock counting (showing ~00:00:05 or similar)
# 3. status_led_on.png  - Status LED (PB0) illuminated
# 4. status_led_off.png - Status LED (PB0) off (2-second toggle)
# 5. task_led.png       - Task LED (PB1) flashing at 5-second interval
# 6. set_mode.png       - SET button pressed, time-set mode active
# 7. inc_mode.png       - INCREMENT button pressed, field changed
# 8. reset.png          - RESET button pressed, clock shows 00:00:00
#
# How to take screenshots in SimulIDE:
#   - Use your OS screenshot tool (e.g., Ctrl+Shift+PrintScr on Ubuntu)
#   - Or SimulIDE → File → Export Image
